# Connect4
Play Connect 4 with another person or with an AI

How to run:
  1) Run ps9pr1
  2) Run ps9pr2
  3) Run ps9pr3
  4) Run ps9pr3
  5) To play with another human player: type in the console connect_four(Player('X'),Player('O'))
  6) To play with an AI: connect_four(Player('X'),AIPlayer('O','LEFT',5)
